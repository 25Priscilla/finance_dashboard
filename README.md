# 💰 Finance Dashboard

A clean and interactive finance dashboard built using React.

## 🚀 Features

- Dashboard overview (Balance, Income, Expenses)
- Transaction management (Add/Delete)
- Category-based pie chart
- Time-based transaction chart
- Role-based UI (Admin / Viewer)
- Insights section (spending analysis)
- Search and filter transactions
- Local storage persistence

## 🛠 Tech Stack

- React.js
- Tailwind CSS
- Recharts

## 💡 Approach

The application is built using a component-based architecture in React. State is managed using useState and useEffect. LocalStorage is used to persist transaction data. Charts are implemented using Recharts for data visualization.

Role-based UI is simulated to demonstrate access control between admin and viewer roles.
